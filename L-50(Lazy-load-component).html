What is Lazy loading component? it is seperate component when we need that component only we call it like when we click login button then only login component call,when we click signup button then only signup component call that means all page not call at a time then our memory and loadin speed increase that time.For example in our project 100 component and when we run our project then all 100 component run at a time then our project get slow at the time we create separate-separate component to load only when user call or click it or work it not all component load.
