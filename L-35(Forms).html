Forms Use:- 1.Submit any kind of information to the database 2.But Angular Cannot connect to the database. 3.So we send data to API, API send our data to the database and API update the database. 4.Example:- a.Login b.Sign Up c.Post d.Google Search is also a type of form

Forms Type in Angular:- 1.Template-driven Form All code we written in our html file(app.componenet.html) these are Template-driven form. That mostly work done in componenet template 2.Reactive Form All code we written in our ts file(app.componenet.ts) these are Reactive form. Mostly work done in componenet class

What we learn in Forms:- 1.How to make Forms 2.How to get Data from Forms 3.How to add validation 4.How to show Error Message 5.How to call API 6.How to show prefill forms

Data Flow with Forms:- Template(.html file) -> Class(.ts file) -> Services(API) -> Database

Interview-Question:- 1.Can Angular directly connect to the database ? No, using API in angular to connect to the database or using backend language like:- php,python
