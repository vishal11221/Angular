What is Inline-style and when we use it ? Inline-style is styline feature that is use for designing our content. When we have less use of css feature then we use Inline-Style. As:- ng g c componentname --inline-style In which only three file is there:- Html file,ts file,spec.ts file , css file is not there because we inline css in ts file

What is Inline-template and when we use it ? Inline-template is angular feature,that is use for we create our html code in ts file,there is no html file.One template feature is available in ts file that help to create your html code inside it. As:- ng g c componentname --inline-template

Can we use Inline-Style and Inline-template in sinlge component ? Yes,it is possible in which only two file is coming,one is ts other is spec.ts files. In which we write our html code and css code in single file only and use it as to declare in app.component.html with html tag.
