What is component? component is building block of code that are seperately used in our Application. For ex:- We have E-Commerce website, so we create login component seperately,signup component seperately,profile component seperately so on, that all component used in app.component.html file as html tag.

Note:- In app.module.ts file your all created component are automatically imported.

How we create component? ng g c componentname

How to use component ? When you want to use component, then you must know the selector name from componentname.component.ts file and use it as html tag.

Can we change the html tag name and use ? Yes,it is possible,simply change the selector name and use.

Can we reuse one component ? Yes,it is possible,but the same funcionality you use other's place also.Simply use this component with html tag.
