Interview-Questions :- 1.What is 404 page ? A 404 page is also known as an “error page” or “Page Not Found” page. This page is the common page,if user enter unknown url or path in ur url then you show this page as Error-page.Like that Page not Found.For Example if any url or any page that not found in your website then you show your 404 page to display the notification to the user that page not found. 2.What is WildCard Route ? The Wildcard Route is basically used in Angular Application to handle the invalid URLs. We use double star (**) symbol to set the path of page-not-found 3.How we use 404 page in Angular ? Simply we create a componet of page-not-found,then set routing of that path is double start that is WildCard then we give wrong url then that page show.
