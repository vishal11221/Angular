Routing is the process to go one view to another view as user perform task in web-app.

Simply we create different componenent like,Home,Login,Register,About,contact-us and set their path in our main componenent html that is app.component.html same as hyperlink as a routerLink then we define that path in app.routing.module.ts and all the given path declare in routerLink.So we simply click on that link and go to that link easily.
