When we setup our angular application then only one Component at the time create that is app component that is default component creating only at the time of setup of Angular application.

When you have more heading in your global component and you want to style this as red to all then that css is global css, so that css apply in style.css file then all the heading automatically take red. And you want the css as component-wise then that css is for component.css like.

What is the priority-list of css in Angular? 1.Inline css, that we assign with tag. 2.Internal css, that you assign in component.html 3.External css, that you assign in component.css 4.Global css, that you assign in style.css for all the component for same styling.
